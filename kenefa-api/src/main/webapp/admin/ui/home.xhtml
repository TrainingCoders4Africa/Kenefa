Kenefa

stands for health in Bambara a largely spoken African language in West Africa.

Kenefa is a centralized data repository system, a wealth of easily accessible, standardized, structured, searchable and downloadable data on doctors, nurses, health agents, hospitals, clinics and any other health facility type private or public for every city and country in Africa. This data intelligence tool aims to support better decision-making and immediate action to remedy continental health issues in Africa.
